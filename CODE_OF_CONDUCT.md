# Code of Conduct

We value the participation of everyone and want all contributors to have a **safe and positive experience**.  

By participating in this project, you agree to abide by this Code of Conduct.

## Our Standards
- Be **respectful**, considerate, and inclusive.
- Avoid harassment, discrimination, or offensive language.
- Respect differing opinions and experiences.

## Enforcement
- Any reports of misconduct should be submitted to the project maintainers.
- Maintainers will review reports and take appropriate action, which may include removal from the project.

## Reporting
If you experience or witness unacceptable behavior, please contact the maintainers immediately.

## Acknowledgment
This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org/), version 2.1.
